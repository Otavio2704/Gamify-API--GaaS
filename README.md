| Mecanismo | Detalhe |
|-----------|---------|
| рҹ”’ **Senhas** | Armazenadas com BCrypt |
| рҹ”‘ **API Keys** | Nunca persistidas вҖ” apenas o hash SHA-256 fica no banco |
| рҹӘҷ **JWT** | `sub` = tenant ID, assinado com HMAC-SHA256 |

| рҹҸў **Multi-tenancy** | Garantido por ThreadLocal вҖ” nenhuma query acessa dados de outro tenant |
